Definiciones y conceptos

Tipo de dato abstracto (TDA)

El tipo de dato abstracto está conformado por: una estructura de datos + un comportamiento

Tipos de TDA
En los tipos de datos abstractos podemos encontrar

  • Pilas
  • Colas
  • Listas
  • Árboles
  • Grafos

Pilas

La pila es un TDA formado por una colección ordenada de elementos homogéneos, dinámica, cuyas inserciones y eliminaciones se realizan por un único extremo llamado TOPE o CIMA.
Último en entrar - Primero en salir

La implementación se puede hacer mediante

  • Vectores
  • Archivos
  • Memoria dinámica (punteros)

Gráficamente: TDA Pila (implementada sobre un vector)

Operaciones con TDA Pila

  • crear(P)   //inicializa una pila
  • apilar(P,X)   //da de alta un elemento a la pila
  • desapilar(P,X)   //elimina un elemento de la pila
  • tamanio(P)   //devuelve el tamaño de la pila
  • pila_vacia(P)   //comprueba si la pila está vacía
  • pila_llena(P)   //comprueba si la pila está llena

Unit: Pilas 
unit u_pilas;
{$codepage utf-8}
interface
USES
    crt;
CONST
     N = 10;
TYPE
    T_dato = record
                   nom: string[40];
                   orig: string[40];
               end;
    T_pila = record
                   tope, tam: 0..N;
                   elem: array [1..N] of T_dato;
             end;
              Procedure crear_pila(VAR P: T_pila);
              Procedure apilar(VAR P: T_pila; X: T_dato);
              Function pila_llena(VAR P: T_pila): boolean;
              Function pila_vacia(VAR P: T_pila): boolean;
              Procedure desapilar(VAR P: T_pila; VAR X: T_dato);
Implementation
              Procedure crear_pila(VAR P: T_pila);
              begin
                   P.tam := 0;
                   P.tope := 0;
              end;
              Procedure apilar(VAR P: T_pila; X: T_dato);
              begin
                   P.tope := (P.tope + 1);
                   P.elem[P.tope] := X;
                   inc(P.tam)
              end;
              Function pila_llena(VAR P: T_pila): boolean;
              begin
                   pila_llena := P.tam = N;
              end;
              Function pila_vacia(VAR P: T_pila): boolean;
              begin
                   pila_vacia := (P.tam = 0);
              end;
              Procedure desapilar(VAR P: T_pila; VAR X: T_dato);
              begin
                   X := P.elem[P.tope];
                   P.tope := (P.tope - 1);
                   dec(P.tam);
              end;
end.

Colas

La cola es un TDA formada por una colección ordenada de elementos homogéneos, dinámica, cuyas inserciones se realizan por un extremo llamado FINAL y las eliminaciones se realizan por otro extremo llamado FRENTE.
Primero en entrar - Primero en salir

La implementación se puede hacer mediante

  • Vectores
  • Archivos
  • Memoria dinámica (punteros)

Graficamente: TDA Cola (implementada sobre un vector)

Operaciones con TDA Cola

  • crearcola(Q)   //inicializa la cola
  • encolar(Q,X)   //da de alta un elemento en la cola
  • desencolar(Q,X)   //elimina un elemento de la cola
  • cola_vacia(Q)   //comprueba si la cola está vacía
  • cola_llena(Q)   //comprueba si la cola está llena
  • cantidad(Q)   //devuelve el tamaño de la cola

Unit: Colas 
unit u_colas;
{$codepage utf-8}
interface
USES
    crt;
CONST
     K = 10;
TYPE
    T_dato = string;
    T_cola = record
                   frente, final: word;
                   elem: array [1..K] of T_dato;
                   tam: word;
               end;
              Procedure crear_cola(VAR Q: T_cola);
              Procedure encolar(VAR Q: T_cola; X: T_dato);
              Procedure desencolar(VAR Q: T_cola; VAR X: T_dato);
              Function cola_llena(VAR Q: T_cola): boolean;
              Function cola_vacia(VAR Q: T_cola): boolean;
Implementation
              Procedure crear_cola(VAR Q: T_cola);
              begin
                   Q.frente := 1;
                   Q.final := K;
                   Q.tam := 0;
              end;
              Procedure encolar(VAR Q: T_cola; X: T_dato);
              begin
                   If Q.final = K then
                   Q.final := 1
                   Else
                   inc(Q.final);
                   Q.elem[Q.final] := X;
                   inc(Q.tam);
              end;
              Procedure desencolar(VAR Q: T_cola; VAR X: T_dato);
              begin
                   X := Q.elem[Q.frente];
                   If Q.frente = K then
                   Q.frente := 1
                   Else
                   inc(Q.frente);
                   dec(Q.tam);
              end;
              Function cola_llena(VAR Q: T_cola): boolean;
              begin
                   cola_llena := (Q.tam = K);
              end;
              Function cola_vacia(VAR Q: T_cola): boolean;
              begin
                   cola_vacia := (Q.tam = 0);
              end;
end.
